This was my first try of a little project to test my understanding of the basics of python.
It is a wordle-inspired game where u have to guess a 5-letter word from a given list. When typing in a word, it will tell u if certain letters are in the word and wether they are on the right spot or not.
Without any visuals (GUI) tho... just text

Features:
  - programm randomly selects a word from the list
  - user has 3 Tries to guess the word
  - different colors for the letters:
      - green: right letter and right position
      - yellow: right letter but wrong position
      - red: wrong letter, not in word

You can run this code by downloading it and open it in a terminal or a programm like VS-Code.

A plenty of limitations like in the original wordle such as handling duplicate letters are not implemented yet.
